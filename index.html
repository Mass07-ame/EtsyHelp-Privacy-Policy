Privacy Policy

EtsyHelp Privacy Policy

Effective date: April 18, 2026

EtsyHelp is a Chrome extension designed to help users filter Etsy search results by updated date or created date. This Privacy Policy explains what data the extension accesses, how it is used, and what is not collected.

1. Overview

EtsyHelp works only on Etsy search pages and related public Etsy listing pages. Its purpose is to help users apply date-based filters, track scan progress, and reuse previous scan results directly inside the browser.

EtsyHelp is designed to minimize data use. It does not require account creation, does not request passwords, and does not sell or share user data with third parties.

2. Information EtsyHelp Accesses

EtsyHelp may access the following limited data only to provide its core functionality:

  • Public content on Etsy search result pages
  • Public content on Etsy listing pages, including the publicly visible “Listed on” date when available
  • User-selected settings inside the extension, such as keyword, date range, scan mode, and pages to scan
  • Extension status information, such as scan progress and last created-scan results

3. Information EtsyHelp Does Not Collect

EtsyHelp does not collect, store, or transmit the following:

  • Personally identifiable information such as name, email address, phone number, or home address
  • Passwords, login credentials, or authentication secrets
  • Payment or financial information
  • Private messages, emails, or personal communications
  • Health information
  • Precise location data
  • Browsing history outside Etsy pages required for the extension to function

4. How Data Is Used

The limited data accessed by EtsyHelp is used only to provide the extension’s core features, including:

  • Filtering Etsy search results based on updated or created date workflows
  • Displaying the floating summary panel and live status information
  • Saving user preferences locally for convenience
  • Saving the last created scan results so users can reuse them later

EtsyHelp does not use user data for advertising, profiling, analytics unrelated to the extension’s functionality, credit scoring, or lending decisions.

5. Local Storage

EtsyHelp uses the browser’s local storage to save settings and working data on the user’s device. This may include:

  • Keyword and selected date/time range
  • Selected mode such as Updated or Created
  • Panel position on the page
  • Current scan status and counters
  • Last created-scan results

This information is stored locally in the user’s browser and is not sent to an external server by EtsyHelp.

6. Website Content Access

EtsyHelp reads publicly available content from Etsy pages only when needed for its filtering features. In Created mode, the extension may request public Etsy listing pages directly in order to read the visible “Listed on” date when that information is available.

This page content is used only to support filtering and scan results inside the extension.

7. Data Sharing and Selling

EtsyHelp does not sell, rent, transfer, or share user data with third parties, except where required by law.

EtsyHelp does not use or transfer user data for purposes unrelated to the extension’s core functionality.

8. Remote Code and External Services

EtsyHelp does not load or execute remote code from third-party servers.

EtsyHelp is intended to operate directly within the browser on Etsy pages and to store working data locally.

9. Data Retention and User Control

Data stored by EtsyHelp remains in the user’s browser until it is cleared by the user, removed by browser settings, or deleted when the extension is uninstalled.

Users can remove local extension data by clearing extension storage through browser tools or by uninstalling the extension.

10. Security

EtsyHelp is designed to minimize data collection and keep working data local whenever possible. However, no software environment can be guaranteed to be completely secure, and users should keep their browser and operating system updated.

11. Children’s Privacy

EtsyHelp is not directed to children and is not intended for use by children under 13.

12. Changes to This Policy

This Privacy Policy may be updated from time to time to reflect changes to the extension or legal requirements. When updated, the latest version should be published at the same policy URL.
